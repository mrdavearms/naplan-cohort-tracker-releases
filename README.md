# Naplan Throughline — releases

Public installers and the auto-update feed for **Naplan Throughline**
(on-device NAPLAN cohort analysis). This repo holds **binaries only** — the
application source is private.

- Download the latest installers from the [Releases](../../releases) tab.
- `latest.json` (attached to each release) is the Tauri auto-update manifest the
  app reads from `releases/latest/download/latest.json`.

Builds are unsigned for v1: macOS → right-click → Open; Windows → More info → Run anyway.
